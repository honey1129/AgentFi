Agent Ownership Runtime

Agent Ownership Workspace

一个把 agent、NFT 所有权、交易、调度和链上同步放进同一块控制台的单机托管面板。

Ownership Layer NFT 定义 agent 控制权

交易、转移、链上同步都会直接影响谁能调度 agent。

Execution Layer Run 入队后再二次校验 owner

队列和 worker 执行前都绑定当前 NFT owner，避免转手后仍被调用。

Market Layer 本地撮合和链上转移统一入口

本地 listing、MetaMask transfer、listener 同步都在同一套 runtime 里闭环。
